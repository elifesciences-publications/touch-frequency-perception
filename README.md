### This code is associated with the paper from Birznieks et al., "Tactile sensory channels over-ruled by frequency decoding system that utilizes spike pattern regardless of receptor type". eLife, 2019. http://dx.doi.org/10.7554/eLife.46510
# touch-frequency-perception
Scripts used to produce and analyse psychophysical data. These data describe how humans perceive frequency of vibration through touch.
